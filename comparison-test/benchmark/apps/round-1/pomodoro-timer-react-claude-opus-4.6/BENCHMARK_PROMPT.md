You are building a Pomodoro Timer app using React.

Build a Pomodoro timer app with:
- A circular progress indicator showing time remaining
- Start/pause/reset controls
- Configurable work (25min) and break (5min) durations
- Session counter (how many pomodoros completed)
- Sound notification when timer ends (use a simple beep via Web Audio API)
- Dark mode toggle
- Clean, minimal UI with good typography

Requirements:
- Single-page app, no backend needed
- Use Vite as the build tool
- All state managed client-side
- Clean, modern UI with CSS (no UI library)
- Dark mode support
- Responsive design
- Put all code in the current directory

Use React 19+ with hooks (useState, useEffect, useMemo, useCallback). Use Vite as the bundler.

Initialize: `npm create vite@latest . -- --template react` then `npm install`
